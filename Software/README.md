Pinky Software
=======

## Install dependencies

See README files in '/ctrl' and '/prcp'

## Run

1. Open terminal and split the window vertically

2. Left terminal:
```
$ ssh robocar@pinky -Y -C
$ cd robocar-pinky/Software/ctrl
$ ./ctrl.sh
```

3. Right terminal:
```
$ ssh robocar@pinky -Y -C
$ cd robocar-pinky/Software/prcp
$ ./prcp.sh
```
